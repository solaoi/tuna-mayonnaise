# TUNA-Mayonnaise

This is a Low Layer CMS.
This application is a tool to generate and serve JSON/HTML with flow model.

## Get Started

[Install here](https://github.com/solaoi/tuna-mayonnaise/releases/tag/v0.0.2-alpha)

### Run Edit-Tool

```
tuna tool
```

### Run API with Tool-Data

```
tuna api
```
